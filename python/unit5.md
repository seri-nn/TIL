### Tuple(튜플)

튜플 : 읽기 전용의 리스트이며, 소괄호로 생성(괄호가 없어도 무방)

- 리스트와 비슷하지만 **값을 읽을 수만 있으며, 수정하거나 새로운 값을 추가할 수 없음**
- 튜플은 중복된 원소가 있을 수 있고 정해진 순서가 있어 순서가 다르면 서로 다른 튜플

e.g. (1,2,3) != (1,3,2)

- 튜플의 항목이 한 개일 때 튜플 뒤에 콤마를 붙여야 함

### 튜플 값 읽기
튜플이름[인덱스] : 특정 항목에 접근

튜플 값을 출력하는 것은 리스트와 동일. 출력하고 싶은 값의 위치를 [] 안에 입력하면 됨

e.g.
```python
Tup1=(10,20,30,40)
print(Tup1[0])
```
---> 10

### 튜플 덧셈과 곱셈
튜플의 덧셈 : 한 개의 튜플 안에 두 개의 튜플 값을 모두 입력하는 것 

튜플의 곱셈 : 곱하는 수만큼 튜플을 반복해 입력하는 것 

### 튜플 & 리스트
튜플에 값을 추가하고 싶을 때는 리스트로 바꿔서 값을 추가한 뒤 다시 튜플로 바꾸는 방법이 있음

e.g.
```python
Tup1=(10,20,30,40)
list1=list(Tup1)
list1.append(50)
Tup2=tuple(list1)
print(Tup2)
```
---> (10,20,30,40,50)

### dictionary (딕셔너리)
딕셔너리 : 2개의 쌍이 하나로 묶이는 자료 구조 

- 중괄호{}로 묶어서 구성 -> {키(key) : 값(value)} 쌍으로 구성

e.g. 딕셔너리변수 = {키1:값1, 키2:값2, 키3:값3...}

- **딕셔너리는 순서가 없음**
- 여러 개의 정보를 하나의 변수로 표현할 때 유용하게 사용됨 eg. 표의 값들을 표현할 때 

### 딕셔너리 정보 추가
- 형식 : 딕셔너리이름[키]=값
- 정보를 추가할 때는 키와 값을 쌍으로 추가해야 함

e.g.
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
x["번호":"010-1234-5678"]
print(x)
```
---> {"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공", "번호":"010-1234-5678"}

- 기존 딕셔너리에 동일한 키가 존재할 경우 값이 변경 = 정보 수정

-> 키는 중복되지 않고 유일함 (값은 중복되어도 상관없음) 

### 딕셔너리 키와 값 
- 딕셔너리 키와 값 삭제 : del(딕셔너리이름[키]) 함수 이용

e.g.
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
del(x["부서"])
print(x)
```
---> {"학번":"20231649", "이름":"조세린"}

- 딕셔너리 값 읽기 : 딕셔너리이름[] 안에 읽고 싶은 값의 키 입력

e.g.
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
print(x["학번"])
```
---> 20231649

### 딕셔너리 함수
- keys() : 딕셔너리의 **모든 키**만 뽑아서 출력

e.g.
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
print(x.keys())
print(list(x.keys()))
```
---> dict_keys(["학번","이름","부서"])

---> ["학번","이름","부서"]  dict_keys가 보기 싫으면 list로 묶으면 되고 list로 묶은 값들은 다 리스트로 출력됨

- get(키) : get 함수에 key 값을 이용해 value 값에 접근 가능

e.g.
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
print(x.get("이름"))
```
---> 조세린 print(x.get("이름")) = print(x["이름"])

- values() : 딕셔너리의 **모든 값**만 뽑아서 출력

e.g.
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
print(x.values())
print(list(x.values()))
```
---> dict_values(["20231640","조세린","데이터사이언스전공"])

---> ["20231649","조세린","데이터사이언스전공"]  dict_vlaues가 보기 싫으면 list로 묶으면 됨

- items() : 튜플 형태로 구하기

e.g. dict_itmes([("학번","20231649"),("이름","조세린),("부서","데이터사이언스전공")])

- in() : 딕셔너리 안에 키가 있는지 확인 -> True/ False로 출력됨

e.g.
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
print("학번" in x)
```
---> True

- **반복문 이용해 딕셔너리에 저장된 키와 값 모두 출력 가능**

e.g.
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
for i in x:
  print(i) # key 값만 출력 
  print(x[i]) # value 값만 출력
  print(i,x[i]) # key, value 값 출력 
```
---> 학번 이름 부서 

---> 20231649 조세린 데이터사이언스전공

---> 학번 20231649 이름 조세린 부서 데이터사이언스전공 

```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
for i in x.keys():
  print(i,x[i]) # key, value 값 출력 
```
```python
x={"학번":"20231649", "이름":"조세린", "부서":"데이터사이언스전공"}
for k,v in x.itmes():
  print(k,v) # key, value 값 출력
``` 
*-> for i in x: = for i in x.keys(): = for k,v in x.items():*

3. 딕셔너리로 성적 합계/평균 구하기
```python
scores={‘김예진’: 90, ‘박영진’: 95, ‘김소희’: 84}
sum=0
for key in scores:
  sum=sum+int(scores[key])
  print("%s : %d" %(key, scores[key]))
avg=sum/len(scores)
print("합계 : %d, 평균 : %.2f" %(sum,avg))
```
---> 김예진 : 90 박영진: 95 김소희 : 84 합계 : 269, 평균:89.67
