### print 함수
- print : 입력한 함수값을 출력해주는 함수

- print 함수에서 사용할 수 있는 서식
    - %d : 정수 **연산 가능**
      ```python
      print("%d" %10)
      ```
      
      숫자를 여러 개 쓰고 싶으면 %()하고 그 뒤에 %d의 갯수만큼 숫자를 써주면 됨
      ```python
      print("%d이고 %d이다" %(50,100))
      ```
    - %f : 실수
      ```python
      print("%f" %3.141592)
      ```
    - %s : 문자열
      ```python
      print("%s" %"데이터")
      ```
    
- 이스케이프 문자 : 입력값 뒤에서 출력값에 변화를 줄 수 있음 
    - \n : 다음 줄로 이동
    - \t : 다음 탭으로 이동
    - \b : 뒤로 한 칸 이동
    - \' : ' 출력
    - \" : " 출력
    - \\ : \ 출력
 
### 데이터형(Data type)
데이터형 : 자료를 기능과 역할에 따라 구분한 것 

크게 숫자형(int, float)과 문자형(str,bool)으로 구분함

print(type()) : type 뒤 괄호가 무슨 데이터형인지 판별해 줌 

- 숫자형
  - int : 정수형
  - float : 실수형 

    정수와 정수의 연산은 정수 / 실수와 실수의 연산은 실수

    정수와 실수의 연산은 실수 / 정수 나누기 정수는 실수
    
- 문자형
  - str(string) : 문자형
  
    문자열은 양쪽을 ""나 ''로 감싸야 함
  
    더하기 연산자(+)를 사용하면 띄어쓰기 없이 다 붙여서 출력
    ```python
    a= 'h'+'ell'+'o'
    print(a)
    ```
    ---> hello

    문자열 곱하기 숫자는 문자를 그 수만큼 반복해서 출력 
    ```python
    a='hi'
    b=3*y
    print(b)
    ```
    ---> hihihi
  
  - bool : 불형
    참이나 거짓만 저장할 수 있는 데이터 형식. 논리형이라고도 함 

### 변수
변수 : 값을 저장하는 메모리 공간

변수 이름은 대소문자 구분, 띄어쓰기 불가, _사용, 영어랑 숫자 혼용해서 지정 가능하지만 영어로 시작해야함

기존에 사용했던 변수가 나오면 이전 값은 어지고 새로운 값으로 덮어씌워짐 

### 변수 값을 입력 받는 함수
- input() 함수 : 키보드로 입력 받도록 도와주는 함수
  input()으로 입력 받는 데이터는 모두 문자열(str)로 인식 = 사칙연산 불가

- 데이터 타입 바꾸기
    - 정수형 -> 실수형 : float로 감싸기
    - 실수형 -> 정수형 : int로 감싸기
    - 문자열 -> 정수형 : int로 감싸기
```python
x= int(input('나눠지는 수:'))
y= int(input('나누는 수:'))
print('%d을 %d로 나눈 몫은 %d입니다.' %(x,y,x//y))
```

### 산술연산자
- '=' : 대입
- '+' : 더하기
- '-' : 빼기
- '*' : 곱하기
- '/' : 나누기
- '//' : 나누기 몫
- '%' : 나머지
- '**' : 제곱

### 비교연산자
- '==' : 같다
- '!=' : 같지 않다
- '>' : 크다
- '<' : 작다
- '>=' : 크거나 같다
- '<=' : 작거나 같다
true / false로 값이 출력됨

### 논리연산자
- and -> 둘 다 참이여야 참
- or -> 둘 중 하나만 참이어도 참
- not -> 참이면 거짓, 거짓이면 참
