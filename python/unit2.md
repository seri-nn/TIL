### if 조건문
if문("만약에~라면") : if문이 참이라면 무엇을 실행하고, 거짓일 때는 아무것도 하지 않는 가장 단순한 형태의 조건문

  - if 뒤에 조건식을 작성 후 콜론으로 if문의 시작을 알림
  - 들여쓰기(4칸)을 통해 if문이 진행 중인지, 종료되었는 지를 파악
  - 들여쓰기가 된 문장들은 다 if 조건문에 포함
  - 들여쓰기 빠져나오기는 shitf + tab

ex1.
```python
password="data"
x=input("비밀번호를 입력해주세요:")
if x == "data":
    print("correct")
```

ex2.
```python
x=int(input("숫자를 입력해 주세요:"))
if x%2==0:
    print("입력하신 숫자는 짝수입니다.")
if x%2==1:
    print("입력하신 숫자는 홀수입니다.")
if x<0:
    print("0보다 큰 숫자 입력")
```

### if ~ else문
if ~ else문 : if일때가 참이 아닐 때 즉, 거짓일 때 else로 출력되는 조건문

*else 뒤 콜론 붙이는 것 주의*

ex.
```python
x=int(input("숫자를 입력해주세요:"))
if x%2==0:
    print("짝수입니다")
else:
    print("홀수입니다")
```

### 중첩 if문
if ~ else ~ if ~ else문 : if문을 한 번 실행한 후 그 결과에서 다시 if문을 실행

-> 조건을 검사하는 과정이 2번인 경우

ex. 서울에 사는 학생 중에서 20세 이상인 학생이 몇 명인지 구하는 프로그램

  - 학생의 주소지가 서울인지 아닌지를 조건문을 통해 확인 후,
  - 그 학생 중에서 나이가 20세 이상인지를 조건문을 통해 다시 확인

### elif문
elif문 : 중첩 if문을 사용할 때 중간의 if~else문을 elif로 줄여서 사용할 수 있음 (실제 코드에서는 이 구문을 가장 많이 사용함)
ex.
```python
x=int(input("숫자를 입력해주세요:"))
if x<0:
  print("0보다 큰 숫자 입력")
elif x%2==0:
  print("짝수")
else:
  print("홀수)
```
